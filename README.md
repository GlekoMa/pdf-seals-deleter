# PDF Seals Deleter

Important: 

1. Need libmupdf 1.23.11, don't use 1.24/1.25;
2. Edit all options of `mupdf-1.23.11-source/include/mupdf/fitz/config.h` to `0` to reduce compile
   time and the size of executable.
